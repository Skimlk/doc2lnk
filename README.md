# doc2lnk

## Overview
`doc2lnk` is a Python tool that converts standard document files into `.lnk` shortcut files. The converted `.lnk` files include the content of the original document appended to the end of the file.

## Requirements
- Python 3.13 or later

---

## Installation

### 1. Create and activate a virtual environment
```bash
python3 -m venv venv
```

**Linux/macOS:**
```bash
source venv/bin/activate
```

**Windows:**
```powershell
.\venv\Scripts\activate
```

### 2. Install dependencies
```bash
pip3 install -r requirements.txt
```
